# The Wild Oasis - 现代化酒店管理系统

The Wild Oasis 是一个功能完整的现代化酒店管理平台，专为酒店运营团队设计，提供高效的客房管理、预订系统、入住/退房处理和数据分析功能。

## 🏨 项目概述

The Wild Oasis 是一个采用 React 和 TypeScript 构建的全栈酒店管理应用，提供直观的仪表板、客房管理、预订系统、入住/退房处理和数据分析等核心功能。应用采用现代化 UI 设计和响应式布局，支持深色/浅色主题切换，为酒店运营团队提供高效的工作体验。

## 🚀 主要功能

### 📊 仪表板 (Dashboard)

- 实时数据概览：预订数量、销售总额、入住数量和入住率
- 销售趋势图表：显示每日销售额变化
- 住宿时长分布：饼图展示客人住宿时长统计
- 今日活动：显示当天到达和离开的客人信息
- 数据过滤：支持按 7 天、30 天、90 天等不同时间段查看数据

### 📝 预订管理 (Bookings)

- 查看所有预订记录
- 详细的预订信息查看
- 预订状态管理（待确认、已入住、已退房）
- 快速入住办理和退房处理
- 预订筛选和排序功能

### 🏠 客房管理 (Cabins)

- 完整的客房信息管理
- 添加新客房和编辑现有客房
- 客房删除功能（带确认机制）
- 客房复制功能
- 客房图片上传和预览

### 📅 入住/退房 (Check-in/Check-out)

- 快速办理入住手续
- 高效退房处理流程
- 入住时早餐服务选择
- 实时更新预订状态

### 👥 用户管理 (Users)

- 用户账户创建和管理
- 用户认证和权限控制
- 用户个人资料管理
- 密码修改功能
- 用户头像上传

### ⚙️ 设置 (Settings)

- 酒店参数配置（最小/最大预订天数）
- 每预订最大客人数量设置
- 早餐价格配置
- 其他系统参数调整

### 👤 账户管理 (Account)

- 个人资料查看和编辑
- 密码修改
- 头像上传和更新

## 🚀 快速开始

### 环境要求

- Node.js 18.0 或更高版本
- npm 或 yarn 包管理器

### 安装与运行

1. **克隆项目**

   ```bash
   git clone https://github.com/your-username/the-wild-oasis.git
   cd the-wild-oasis
   ```

2. **安装依赖**

   ```bash
   npm install
   ```

3. **开发环境启动**

   ```bash
   npm run dev
   ```

   应用将在 http://localhost:5173 启动

4. **构建生产版本**

   ```bash
   npm run build
   ```

5. **预览生产构建**
   ```bash
   npm run preview
   ```

## 📈 仪表板功能详解

### 统计数据 (Stats)

1. **预订数量** - 统计期间内的总预订数
2. **销售总额** - 统计期间内的总收入（人民币）
3. **入住数量** - 统计期间内的已确认入住数
4. **入住率** - 客房利用率百分比

### 销售图表 (Sales Chart)

- 折线图展示每日销售额趋势
- 支持切换查看入住数或销售额
- 响应式设计，适配不同屏幕尺寸

### 住宿时长图表 (Duration Chart)

- 饼图直观展示客人住宿时长分布
- 分类统计：1 晚、2 晚、3 晚、4-5 晚、6+晚
- 支持深色/浅色主题自动适配

### 今日活动 (Today Activity)

- 实时显示当天到达和离开的客人
- 提供快捷操作按钮，直接办理入住/退房
- 包含客人基本信息和客房信息

## 🎯 使用场景

1. **酒店前台** - 快速查看当日预订和入住情况，高效办理入住/退房
2. **运营经理** - 监控酒店业绩和趋势分析，制定运营策略
3. **客房管理** - 管理客房状态和信息，维护客房数据
4. **财务人员** - 跟踪收入和预订数据，进行财务分析
