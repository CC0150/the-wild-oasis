# The Wild Oasis - 酒店管理系统

The Wild Oasis 是一个现代化的酒店管理应用程序，专为酒店运营团队设计，帮助他们高效管理客房、预订、入住和财务数据。

## 🏨 项目概述

The Wild Oasis 是一个功能完整的酒店管理平台，提供了直观的仪表板、客房管理、预订系统、入住/退房处理和数据分析等功能。该应用使用 React 和 TypeScript 构建，采用现代化的 UI 设计和响应式布局。

## 🚀 主要功能

### 1. 仪表板 (Dashboard)

- 实时数据概览：预订数量、销售总额、入住数量和入住率
- 销售趋势图表：显示每日销售额变化
- 住宿时长分布：饼图展示客人住宿时长统计
- 今日活动：显示当天到达和离开的客人信息
- 数据过滤：可按 7 天、30 天、90 天等不同时间段查看数据

### 2. 预订管理 (Bookings)

- 查看所有预订记录
- 预订详情查看
- 预订状态管理（待确认、已入住、已退房）
- 预订操作：入住办理、退房处理、删除预订
- 预订筛选和排序功能

### 3. 客房管理 (Cabins)

- 查看所有客房信息
- 添加新客房
- 编辑客房详情
- 删除客房
- 客房状态管理
- 客房筛选和排序功能

### 4. 入住/退房 (Check-in/Check-out)

- 快速办理入住手续
- 办理退房手续
- 更新预订状态
- 入住时可选择添加早餐服务

### 5. 用户管理 (Users)

- 创建新用户账户
- 用户认证和授权
- 用户个人资料管理
- 密码修改
- 用户头像上传

### 6. 设置 (Settings)

- 更新酒店设置（如最小/最大预订天数、每预订最大客人数量、早餐价格等）

### 7. 账户管理 (Account)

- 用户个人资料管理
- 密码修改
- 头像上传

## 🛠 技术栈

### 前端技术

- **React 18** - 用于构建用户界面
- **TypeScript** - 提供类型安全
- **React Router v7** - 路由管理
- **Styled Components** - CSS-in-JS 样式解决方案
- **Recharts** - 数据可视化图表库
- **React Hook Form** - 表单处理
- **React Icons** - 图标库
- **Zustand** - 状态管理（深色模式）
- **React Hot Toast** - 通知系统

### 数据管理

- **TanStack React Query** - 服务器状态管理
- **Supabase** - 后端即服务（数据库和认证）

### 开发工具

- **Vite** - 构建工具
- **ESLint** - 代码质量检查
- **Date-fns** - 日期处理库

## 📈 仪表板功能详解

### 统计数据 (Stats)

1. **预订数量** - 统计期间内的总预订数
2. **销售总额** - 统计期间内的总收入
3. **入住数量** - 统计期间内的已确认入住数
4. **入住率** - 客房利用率百分比

### 销售图表 (Sales Chart)

- 显示每日销售额趋势
- 可切换查看不同数据（入住数、销售额）
- 支持深色/浅色模式

### 住宿时长图表 (Duration Chart)

- 饼图展示客人住宿时长分布
- 分类统计（1 晚、2 晚、3 晚、4-5 晚等）
- 支持深色/浅色模式

### 今日活动 (Today Activity)

- 显示当天到达和离开的客人
- 支持直接办理入住/退房手续

## 🎯 使用场景

1. **酒店前台** - 快速查看当日预订和入住情况
